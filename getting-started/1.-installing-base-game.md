# ① 安裝 《戰地風雲2142》 主遊戲

在這個教程中，我們將引導您使用多個線上資源來下載遊戲，以及正確的安裝程序。如果您遇到任何問題或有任何疑問，歡迎加入我們的 [Discord 伺服器](https://discord.gg/DaMVNknVnV)以獲取協助。

{% hint style="warning" %}
在下載或安裝遊戲之前，別忘了查看[系統需求](1.-installing-base-game.md#xi-tong-xu-qiu)。
{% endhint %}

如果你需要我們的協助的話，歡迎加入我們的 [Discord 伺服器](https://discord.gg/DaMVNknVnV)或[聯絡我們](../others/contact-us.md)。
{% endhint %}

## 步驟

1. 從 [下載](1.-installing-base-game.md#downloads) -> 主要來源 下載 **BF2142\_Setup.rar**。
2. 在你的桌面上建立一個新的文件夾。
3. 使用 [文件壓縮工具](#user-content-fn-1) 打開 **BF2142\_Setup.rar**[^1]。
4. 將 .rar 文件中的所有項目拖放到新文件夾中。
5. 在文件夾中找到 **setup.exe**。雙擊運行安裝程序。
6. 當彈出對話框 **“是否允許此應用對您的設備進行更改？”** 時，點擊 **是**。
7. 按照嚮導的指示完成安裝。這裡提供了一些[重要備註](1.-installing-base-game.md#wizard-remarks)。
8. 當安裝完成時，取消選中 **“查看自述文件”** 選項，完成嚮導。[**請勿啟動遊戲**](#user-content-fn-3)[^3]。
9. 繼續[對遊戲進行 v1.51 版本的補丁更新](2.-installing-v1.51-patch.md)。[**請勿嘗試略過此步驟**](#user-content-fn-4)[^4]。

## 巫師備註

1. 如果你看到提示 <mark style="color:blue;">選擇安裝語言</mark> (英文: Select installer language) -> <mark style="color:blue;">從下列選項中選擇安裝語言</mark> (英文: Select a language from the list)，請參考以下交叉參考列表選擇你偏好的語言。
   * 中文 (繁體) -> Chinese (Traditional)
   * 丹麥語 -> Danish
   * 日語 -> Japanese
   * 西班牙語 -> Spanish
   * 法語 (標準) -> French (Standard)
   * 波蘭語 -> Polish
   * 芬蘭語 -> Finnish
   * 俄語 -> Russian
   * <mark style="color:blue;">英語 -> English</mark>
   * 捷克語 -> Czech
   * 荷蘭語 -> Dutch
   * 瑞典語 -> Swedish
   * 意大利語 -> Italiano
   * 德語 -> German
2. 如果你看到彈出 <mark style="color:blue;">This game has only been tested on WinXP(32bit)</mark>，點擊 <mark style="color:blue;">Confirm</mark> 安全地忽略此警告，因為遊戲在大多數[^10] Windows 平台上運行良好。
3. 當你被要求提供序列號時，只需從以下 [<mark style="color:blue;">標準版</mark>](#user-content-fn-11)[^11] 中選擇一個密鑰即可通過提示。更多密鑰可在此處找到 [連結](https://mozziefiles.wixsite.com/bf2142)。
   * SDV5-JZDE-L6B8-P2Q4-1911
   * SVMV-LBNV-W7W5-J3X4-1911
   * E6HH-DWUG-U8X1-R8F0-1911
   * UTFP-4DMN-D2U0-X3T3-1911​
   * TRCL-YBIJ-D4H4-Y3O5-1911
4. 強烈建議你將遊戲安裝到 [默認路徑](#user-content-fn-12)[^12]，但將其安裝到其他位置也是可以的。
5. 強烈建議你安裝 <mark style="color:blue;">英文</mark> 版本的遊戲，以便利用 Project Remaster mod 提供的 <mark style="color:blue;">HD English fonts</mark>。
6. BattleCammo 是一個語音聊天服務，允許你在遊戲中進行語音聊天。信不信由你，[你不會經常使用它](#user-content-fn-13)[^13]。你可以通過點擊 <mark style="color:blue;">Cancel</mark> 按鈕來跳過配置它。
7. GameSpy Comrade 是一個提供朋友、統計數據和即時消息等功能的社區遊戲應用程式。不幸的是，由於 [GameSpy shutdown](https://en.wikipedia.org/wiki/GameSpy#Shutdown) 於2014年6月關閉，它已不再可用。選擇 <mark style="color:blue;">No, install GameSpy Comrade later</mark> 選項以跳過安裝它。
8. 由於 [GameSpy shutdown](https://en.wikipedia.org/wiki/GameSpy#Shutdown) 於2014年6月關閉，EA Master Server 已不再存在。點擊 <mark style="color:blue;">REGISTER LATER</mark> 以跳過註冊 EA 帳戶。
